# IPTV
此项目通过其它直播源抓取集成一个总直播源，直播内容通过demo自定义进行运行爬取。每天自动更新直播，提供在线地址，可以直接使用各大播放平台。M3U配置peg和台标，支持IPV6/4双栈访问!
> 声明: 所有播放源均收集于互联网，仅供测试研究学习，`不得商用!`
